Урок 5. Задача: научиться работать с Cursor, читать и менять код через AI. Добавить 2-3 улучшения MVP.

## Промпты, использованные в Cursor

### Промпт 1: Понимание проекта

> Объясни, что это за проект, из каких файлов он состоит простыми словами

### Промпт 2: Комментирование кода

> Добавь комментарии ко всем важным частям кода, чтобы было понятно, что и где

### Промпт 3: Тёмная тема

> Добавь переключатель тёмной темы в header. При нажатии должен меняться класс на body. Создай CSS-переменные для цветов и два набора значений: для светлой и тёмной темы. Сохраняй выбор темы в localStorage. Все остальное оставь прежним.

### Промпт 4: Подтверждение отмены

> Добавь модальное окно подтверждения перед отменой бронирования. Модалка должна показывать название ресурса, дату и время. Кнопки: «Отменить бронь» (красная) и «Назад» (серая). Стили модалки добавь в style.css. Всё остальное оставь прежним.

### Промпт 5: Сортировка бронирований

> Добавь select для сортировки бронирований над списком. Три варианта: по дате создания (по умолчанию, новые первые), по дате бронирования (ближайшие первые), по статусу (сначала активные). Обнови renderBookings() для поддержки сортировки. Всё остальное оставь прежним.

Добавлены 3 улучшения: тёмная тема, подтверждение отмены, сортировка бронирований. Подробности в `notes/hw6-cursor-improvements.md`.
