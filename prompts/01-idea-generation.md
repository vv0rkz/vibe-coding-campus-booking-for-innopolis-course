# Промпт 1 — Генерация идей для MVP

## Контекст
Урок 1. Задача: сгенерировать 3 идеи для курсового проекта по дисциплине «Программирование без кода».

## Промпт

> Ты — продуктовый менеджер с опытом в EdTech и университетских сервисах.
>
> Предложи 3 идеи для MVP, которые можно реализовать за учебный курс с помощью no-code / AI-инструментов (Bolt.new, Lovable, Cursor, Supabase).
>
> Требования к идеям:
> - Решают реальную проблему студентов или сотрудников университетского кампуса
> - Имеют понятную целевую аудиторию
> - MVP можно собрать за 4–6 недель командой из 2–3 человек
> - Включают минимум: каталог/список, форму ввода, личный кабинет
>
> Для каждой идеи укажи: название, проблему, ЦА, ключевые 3 функции, почему это реально сделать как MVP.

## Результат
ИИ предложил 3 варианта, из которых был выбран **«Сервис бронирования ресурсов кампуса»** как наиболее структурированный и полезный для повседневной жизни кампуса.
