# Промпт 4 — Генерация лендинга для MVP

## Контекст
Урок 3. Задача: создать одностраничный лендинг с формами, интерактивными сценариями и сохранением данных в localStorage.

## Промпт

> Ты — фронтенд-разработчик. Создай одностраничный лендинг для сервиса бронирования ресурсов университетского кампуса «CampusBook».
>
> Стек: чистый HTML + CSS + JavaScript (без фреймворков).
>
> Страница должна содержать:
> 1. **Header** — логотип, навигация (якорные ссылки)
> 2. **Hero-секция** — заголовок, подзаголовок, CTA-кнопка «Забронировать»
> 3. **Блок «Как это работает»** — 3 шага с иконками
> 4. **Каталог ресурсов** — карточки с фото (unsplash), названием, типом, описанием, этажом, вместимостью. Кнопки-фильтры по типу (переговорные, коворкинги, оборудование, консультации)
> 5. **Форма бронирования** — поля: имя, выбор ресурса (select), дата, время начала/конца (select с шагом 30 мин), цель. Валидация: время начала < конца, проверка конфликтов
> 6. **Секция «Мои бронирования»** — список бронирований из localStorage, кнопка «Отменить»
> 7. **Footer**
>
> Данные бронирований хранить в localStorage (ключ `campusbook_bookings`).
> Дизайн: современный, минималистичный, синяя цветовая палитра, адаптивный.
> Код разделить на 3 файла: index.html, style.css, script.js.

## Результат
Получен полный рабочий лендинг с двумя интерактивными сценариями (бронирование + управление бронями), фильтрацией каталога и сохранением данных в localStorage.
